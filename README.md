# Personal Portfolio Website

A lightweight, clean, and fully responsive personal portfolio website built to showcase my web development journey, technical skills, and recent projects.

## 🚀 Tech Stack
*   **Frontend:** HTML5, CSS3, Vanilla JavaScript
*   **Framework CSS:** Bootstrap 5
*   **Deployment:** Vercel

## ✨ Features
*   **Responsive Design:** Fully optimized for mobile, tablet, and desktop screens.
*   **Dynamic Project Loading:** Projects and experiences are rendered dynamically using JavaScript for easier maintenance.
*   **Clean Architecture:** Structured assets and components for better readability.
